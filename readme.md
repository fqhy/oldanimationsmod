## OldAnimationsMod

OldAnimationsMod based off v2.7.1 without cosmetics and external connections to spiderfrog's websocket

- Removed most of obsolete and dead code, remains of some sort of ESP, useless util classes
- Removed all cosmetics related classes
- Removed all cosmetics assests
- Removed all connection classes
- Remade overly complicated config system (~550 lines down to ~130)
- Replaced a few buttons and strings names here and there

Original OAM is 1.9MB, while my version is only 655KB.
